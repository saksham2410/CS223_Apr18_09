Implemented an online examination portal for students to give examination. 
There is a list of four courses currently - English, Chemistry, Maths, Physics.
The professors can add and modify the questions, set the positive and negative marks for the questions, can set their administrator passwords. view student details and results. The faculty(Admin) can edit student details too. 
New students are added according to their first and last name according to their Date of Birth. Sequential Application numbers are assigned to the students. The students can appear for a test by entering their applicant number and their date of birth, and can view their results

1.All data files and source files should be extracted in same folder so that paths need not be changed much
2.Compile and run the file main.cpp
3.Password of Admin is "password"
4.Add questions and students in the database if required
